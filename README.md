# Typescript API Starter

For personal use!

## Development

Create `.env` file in the root with this contents:

```
NODE_ENV=development
PORT=4003
```

Then:

``` bash
$ yarn
$ yarn dev
```

## API Documentation

Head to [http://localhost:4003/api-docs](http://localhost:4003/api-docs) to read the API documentation.
